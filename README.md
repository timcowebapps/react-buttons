### React-Buttons

Микрокомпонент базовой кнопки.

### API

````jsx
var btnStyles: any = require('timcowebapps-react-buttons-dir/button.scss');

<Button schema={{
	properties: {
		tag: "button",
		classes: btnStyles.tg_btn
	}
}}>Button content</Button>
````

| Property       | Type               | Default     | Description |
| :------------- | :----------------- | :---------- | :---------- |
| `schema`       | `any`              | `-`         |             |
